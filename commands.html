Configuration of the Event
To create an event MANUALLY, type get-config in your server and copy/paste it in a text file.
Find "events":0
The synthax of an event is: {"important":{"events":[{"type":"type_name","type_parameter":parameter,"repeat":number,"commands":["your_command","your_command"]}]}}
Type of events"type":"location"
or "score"
or "time"
or "craft"
or "inventory"
or "day"
or "kill"
or "battle-royale"
or "murder"
or "locationInventory" 		"location"
This type triggers an event when a player is at a given position.
Ex: When a player goes on the position 50 50, he wins 50 points and a message appear for everyone.
The parameters of this type are x and y.
Usage: {"type":"location","x":50,"y":50,"repeat":0,"commands":[...]}

"score"
This type triggers an event when a player reaches a given score.
Ex: When a player reach 1000 points, everybody is teleported to the area 50 50 10 10 and the attack is disabled.
The parameter of this type is score.
Usage: {"type":"score","score":1000,"repeat":0,"commands":[...]}

"time"
This type triggers an event at a given time.
Ex: When the game has start since 60s, new rules are displayed (with the command welcome).
The parameter of this type is time.
Usage: {"type":"time","time":60,"repeat":0,"commands":[...]}

"craft"
This type triggers an event when a player craft a specific item.
Ex: When a player craft a boat, he is teleported at a position.
The parameter of this type is item.
Usage: {"type":"craft","item":"boat","repeat":0,"commands":[...]}

"inventory"
This type triggers an event when a player has a specific item in his inventory.
Ex: When a player finds a wood helmet in a treasure chest, he wins 5000 points.
The parameters of this type are item, amount.
Usage: {"type":"inventory","item":"wood_helmet","amount":1,"repeat":0,"commands":[...]}

"day"
This type triggers an event when a player survives a specific number of days.
Ex: When a player survive 3 days, he receives 10 cakes.
The parameter of this type is days.
Usage: {"type":"day","day":3,"repeat":0,"commands":[...]}

"kill"
This type triggers an event when a player kills a certain number of people.
Ex: Once a player kills 10 people, he is kicked.
The parameter of this type is kill.
Usage: {"type":"kill","kill":10,"repeat":0,"commands":[...]}

"battle-royale"
This type creates a Battle Royale mode. That means that the event is triggered when there is only one player left alive
The type does not use parameters, and does not use the "repeat" parameter.
Usage: {"type":"battle-royale","commands":[...]}
To use this event, you also have to use the command "battle-royale" to start the battle royale.

"murder-mode"
This type creates a Murder mode. That means that you spawn either as an innocent, as a sherif or as a murderer
The type use the parameters , and does not use the "repeat" parameter.
Usage: The parameters of this type are duration, commandsFail, commandsInnocentWin and commandsMurderWin
To use this event, you also have to use the command "murder-mode" to start the mode.
"locationInventory"
This type triggers an event when a player has a specific item in his inventory in a specific location
This event works exactly like a combined "inventory" and "position" command
The parameters of this type are item, amount, x, y
Usage: {"type":"locationInventory","item":"wood_helmet","amount":1,"x":5,"y":15,"repeat":0,"commands":[...]}
Repeat the event "repeat":number, The parameter "repeat" determines how long it takes for the event to be repeated.
0 means the event will not repeat.
A number greater than 0 means the event will repeat every x seconds.
Usage: "repeat":number .
Example: "repeat":30 The event will repeat every 30s each time a player re-active the event. Example, each time someone gets 500 stones, he is teleported to the position 100 150. He will be teleported again at the position 100 150 only if he reaches 500 stones again after 30s

Add Commands to the event "commands":["first_command", "second_command", etc] The parameter "commands" determines what will happen once the event is triggered.
The commands can be every commands in game such as : "give", "teleport", "message", "give-score" etc.

Add many events You can put as many events you want The synthax to put many events is "events":[{"your_event"},{"your_event"},...,{"your_event"}]
Here is an example of several events :
"events":[{"type":"time","time":60,"repeat":0,"commands":["spawn-area 20 20 10 10","teleport-all 50 50","battle-royale"]},{"type":"battle-royale","commands":["message $name is the winner!","welcome $name is the winner of the last game\n with a total of $kill kills in $day days", "clean-inventory-all", "give-score $id 100000","teleport-all 50 50", "battle-royale"]},{"type":"location","x":55,"y":55,"repeat":800,"commands":["give $id sword 1"]}]
Explanation :
First event -> "time" :
The time is set to 60, which means that the event will be triggered after 60 seconds. The "repeat" is set to 0 which means the event will not repeat.
What will happen after 60 seconds? The new players will spawn in the area 20 20 10 10. The players already in game will be teleported to the position 50 50 and the "battle-royale" command enable the start of the battle royale game.
Second event -> "battle-royale":
Which means the event will be triggered when, of all the players present at the time the command "battle royale" was launched, only one remains.
What will happen when only one player left? A message will appear to everyone :"(the_name_of_the_winner) is the winner!". A text will display in the bottom left the message (the_name_of_the_winner) is the winner of the last game, with a total of (number_of_kills) kills in (number_of_days) days. Then, all inventory will be cleaned, the winner will receive 100000 and every players will be teleported to the position 50 50 again for a new battle royale (with the "battle-royale" command again)
Third event -> "location":
Which means the event will be triggered once a player is in a specific position. Here, it is the position 55 55 and this event will repeat every 800 seconds for each player.
What will happen when a player goes at this position ? He will receive a sword in his inventory. He will be able to receive a sword again only after 800 seconds if he goes once again at this postion. Same for every players.

Synthax rules : Put " " between each words. Only numbers do not need to be between " ".
Remember to always close "events": with ]}]
